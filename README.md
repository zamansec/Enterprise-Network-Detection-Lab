# Enterprise Network Security Monitoring Lab

> Enterprise-grade Network Security Monitoring (NSM) platform built using Splunk Enterprise, Zeek, and Suricata for centralized log collection, network visibility, intrusion detection, threat hunting, and security analytics.

---

# Overview

The Enterprise Network Security Monitoring Lab is a practical Security Operations Center (SOC) project designed to simulate enterprise-level network monitoring and threat detection.

The primary objective of this project is to build a centralized monitoring platform capable of collecting, analyzing, and investigating network telemetry generated by enterprise environments. By integrating Splunk Enterprise with Zeek and Suricata, the lab provides comprehensive visibility into network activity and demonstrates how modern SOC teams detect and investigate potential security threats.

This repository serves as a continuously evolving cybersecurity project focused on Security Monitoring, Detection Engineering, Threat Hunting, and SOC Operations.

---

- Build an enterprise-grade Network Security Monitoring platform.
